# Онлайн Библиотека - Laravel

Приложение для управления онлайн-библиотекой с авторами и книгами.

## Установка

1. Скопируйте все файлы в ваш Laravel проект
2. Настройте базу данных в `.env`
3. Выполните миграции:
\`\`\`bash
php artisan migrate
\`\`\`

4. Заполните базу данных тестовыми данными:
\`\`\`bash
php artisan db:seed
\`\`\`

5. Установите Laravel Breeze для авторизации (опционально):
\`\`\`bash
composer require laravel/breeze --dev
php artisan breeze:install blade
npm install && npm run dev
php artisan migrate
\`\`\`

6. Запустите сервер:
\`\`\`bash
php artisan serve
\`\`\`

## Структура

### Таблицы:
- **authors** - авторы книг
- **books** - книги
- **author_book** - связь Many-to-Many между авторами и книгами

### Маршруты:
- `/books` - список всех книг
- `/books/{id}` - детали книги
- `/authors` - список всех авторов
- `/authors/{id}` - детали автора

### Модели:
- `Author` - модель автора с отношением `belongsToMany(Book::class)`
- `Book` - модель книги с отношением `belongsToMany(Author::class)`

## Функционал

- Просмотр списка книг с авторами
- Просмотр списка авторов с их книгами
- Детальная информация о каждой книге
- Детальная информация о каждом авторе
- Many-to-Many связь между авторами и книгами
- Встроенная авторизация Laravel
